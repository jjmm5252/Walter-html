# DESARROLLO DEL SITIO: SOLUCION A CASO DE ESTUDIO GIMNASIO

- 1 : Creamos el directorio para el proyecyo: gimnasiov1
- 2 : de acuerdo al sitemap para cada pagina cree un archivo html asi:
 - idex.html: pagina principal del sitio, punto de entrada
 - contacto.html: informacion de contacto, google maps
 - horarios.html: fotos de las intalaciones, catalogo de clases, horarios
 - nosotros.html: historia del gym, staff de coach
 - precios.html: tabla de precios, promociones, comparativa de planes
- 3 : creamos subdirectorios ASSETS (ORGANIZAR LOS ARCHIVOS DE SOPORTE DEL PROYECTO) imagenes, videos, js, css, para cada tipo creamos un subdirectorio
- 4 : creacion del menu: usamos listas desordenadas UL y en cada li pongo la opcion, el atajo rapido es li*5
- 5 : Dentro de cada li pongo una etiqueta ancla, anchor hiperenlace que enlazara a la pagina correspondiente
- 6 : atajo para seleccionar todo el texto Ctrl + A
